iNFekt - FAQ

The rendered mode is slow.

Please try to disable "Hardware (GPU) acceleration" in the preferences' Advanced section.

How do I open the menu?

Simply press the Alt key on your keyboard. You can also permanently show it using the menu button from iNFekt's toolbar.

I get an error message "Side-by-side configuration is incorrect" or "MSVCR110.DLL is missing" when I try to start the application.

You need to download and install the Visual C++ Redistributable for Visual Studio 2012 Update 4 from Microsoft. Alternatively, you can get the iNFekt setup file which will download and install the required files automatically. The DLLs are also shipped with the portable versions.

Earlier Versions: up to and including version 0.8.5 the Visual C++ 2010 runtime was required:

  • Microsoft Visual C++ 2010 SP1 Redistributable Package (x86)
  • Microsoft Visual C++ 2010 SP1 Redistributable Package (x64)

Where can I get a Windows 2000 compatible version?

You can use iNFekt 0.7.9 (32-bit).

Where can I get a Windows XP 64-bit compatible version?

You can use iNFekt 0.8.5 or any 32-bit version.

Using the auto update says "The program can't start because MSVCP110.dll is missing from your computer." on Windows 7 or 8 64-bit

Please download and install vcredist_x86.exe (not x64) from: here.
